InputModel(Byte[])

InputModel representing data contained within an .m64 file.

Declaration
public InputModel([NotNull] byte[] input)
Parameters
Type Name Description
System.Byte[] input

A 4-byte value containing X and Y analogue positions, and the XOR of buttons pressed. The first two bytes are the XOR of the buttons, followed by the X and Y inputs represented by 1-byte each.

Given the input 0xC0182541, this can be seen as:

  • Button Flags2-bytes = C0 18
  • X Analogue1-byte = 25
  • Y Analogue1-byte = 41
When reading from an .m64 file from offset 0x400, 4-bytes at a time, the following code works if the hex input is NOT REVERSED.
